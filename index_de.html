Installation

Mobile Version

  1. auf hertuntergelade Datei rechtsklicken und "Archiv hierher auspacken", "Hier extrahieren" o. Ä. auswählen
  2. den Ordner QeoDart öffnen
  3. auf die Datei QeoDart (doppel-)klicken
Hinweis: Die mobile Version speichert sämtliche Einstellungen in der Datei QeoDart.conf und hinterlässt ansonsten keine weiteren Spuren. Wenn diese Datei gelöscht wird, verh&auuml;llt sich die mobile Version genauso wie die normale Version, d. h. die Einstellungen werden im Benutzerverzeichnis gespeichert. Außerdem können Sie problemlos z. B. die mobile Linux-Version und die mobile Windows-Version parallel mit denselben Einstellungen nutzen. Kopieren Sie einfach dazu die Datei QeoDart der Linux-Version in den Ordner der mobilen Windows-Version
