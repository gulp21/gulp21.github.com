Installation

Mobile Version

  1. right click on downloaded file and click on "Extract Archive Here", "Extract All" or the like
  2. open the folder QeoDart
  3. (double) click QeoDart
N.B.: The mobile version saves every setting in the file called QeoDart.conf and does not touch any other files. When this file is deleted, the mobile version behaves like the normal version, i. e. the settings are saved in the user's home directory. Furthermore, you can combine e. g. the mobile version for Linux and Windows so that you can use them with the same settings. Just copy the file QeoDart of the Linux version to the folder of the mobile Windows version.
